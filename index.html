SEO y Meta Etiquetas, Información y conceptos

¿Qué es SEO?

También conocido como la optimización para motores de búsqueda, por su significado al español, son procesos y normas que optimizan un sitio web y su contenido para mejorar la visibilidad y la posición en los resultados de búsqueda. Corporaciones como Google y Yahoo! implementan procesos para organizar la cantidades masivas de páginas web que se crean día con día.

¿Cómo hacer una web optimizada para SEO?

Los procesos más usados involucran el rastreo, la indexación y la publicación de resultados de búsqueda.

  • Rastreo: proceso en el que los motores de búsqueda recorren la web para explorar nuevas páginas, usando programas que siguen los enlaces que existen entrer las páginas.
  • Indexación: cuando una página ha sido rastreada, los motores instentan cmprender su contenido, tanto lo que dice el texto, como las etiquetas, atributos, palabras clave, etc. Si el motor de búsqueda considera que la página cumple con ciertos criterios, agrega información relevante en una base de datos de índices de páginas web.
  • Publicación de resultados de búsqueda: cuando un usuario hace una consulta, el motor de bpusqueda usa el índice para encontrar y mostrar las páginas web más apropiadas.

Existen muchas maneras de optimizar una página web para SEO, y cada una tiene varias formas de conseguir nuestro objetivo. Algunas técnicas para optimizar una pagina web son las siguientes:

  1. Organizar la información de fomar lógica
  2. Perfeccionar el contenido de la página
  3. Usar palabras clave sobre el tema
  4. Optimizar los títulos de la página web
  5. Usar etiquetas meta
  6. Oprtimizar la imagenes
  7. Utilizar enlaces externos

Definición y función de las Meta Etiquetas

Una metaetiqueta es una etiqueta tipo HTML que ofrece información sobre una página web. Lo que las diferencia de una etiqueta HTML común, es que estas no afectan la presentación de la página. Puede proporciar infromación como el autor de la página, el tema que trata, las palabras claves utilizadas y demás información relevante a la página solicitada.

Algunos tipos de metaetiquetas son los siguientes:

  1. Metaetiquetas de descripción
  2. Metaetiquetas de palabras clave
  3. Metaetiquetas de autor
  4. Metaetiquetas de robots
  5. Metaetiquetas de contenido
  6. Metaetiquetas de redirección

Meta Etiquetas y SEO

Las metaetiquetas son fundamentales para la optimización para motores de búsqueda, ya que estos usan esta información para la contrucción de sus índices, y proporcionan una mejor idea de lo que trata la página y su información.

Es gracias a las metaetiquetas que se puede mejorar la visibilidad de nuestro sitio web, ya que influyen con la precisión con la que los motores clasifican y muestran las páginas
